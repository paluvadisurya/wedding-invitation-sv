Optional background music goes here.

Drop a licensed audio file (a soft nadaswaram or veena loop suits the
theme beautifully) as theme.mp3, then set in config.json:

  "music": { "src": "assets/audio/theme.mp3", "title": "…", "credit": "…" }

A mute/unmute control appears automatically in the top bar.
Audio never autoplays — guests opt in with a tap.
